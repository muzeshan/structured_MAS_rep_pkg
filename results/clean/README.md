# Clean-run results

`record_level_results.csv` contains one derived row for each of the 174 clean workflow outputs. The remaining CSVs reproduce RQ1 efficiency/effectiveness, pair correctness, difficult cases, incorrect cases, and structured RQ2 lineage.

The original 174 stage-level clean JSONL records were not among the files available when this public repository archive was assembled. Place them under `results/clean/raw/` when available. Their absence does not prevent recomputation of the paper's clean aggregate tables from the included CSVs, but it limits independent re-parsing of the clean natural-language handoffs.
